Terms:

Click on a term to see definition.

  • JavaScript(JS)Most common web programming language used to make programs interactive.
  • OperatorsA character that indicates an action (ex: +,-,*, etc.)
  • VariablesContainer meant to store values assigned or label data.
  • Variable naming conventionsUse Lower camel case; start with a lowercase letter and capitalize any following words.
  • FunctionsPerforms an action and returns a result.
  • MethodsAn action that sends message to a piece of data and returns its response.
  • ArgumentsInformation passed to the method.
  • ParametersA variable that holds an argument passed into methods.
  • ReturnMethod's response.
  • Chaining methodsImmediately calling a method on result of another method.
  • StringsDatatype that represents entered text within quotation marks.
  • BooleansTrue or False, not strings.
  • UndefinedObject or variable with no defined value.
  • NaNNot a number.
  • EscapeUsing \\ to allow strings special characters.
  • AlertPrints message as a popup.
  • CommentsAnything following // serves as hidden text for programmers.
  • jQueryA library for adding interactability to webpages using javascript.
  • AttributesQualities assigned to elements on html page.
